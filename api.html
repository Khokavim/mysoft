We can perform either a standard check or an express check with two different workflows.

Standard checks have a workflow that sends a form to the applicant to complete.

All express checks have the following basic workflow:


1. An applicant is created

2. Documents and images are uploaded to the applicant object (if required).

3. Check is constructed consisting of one or more reports.
